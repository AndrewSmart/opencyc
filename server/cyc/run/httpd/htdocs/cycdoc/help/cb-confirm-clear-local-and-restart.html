Since clearing the local queue will cause all operations in the queue to be discarded without being processed, and may result in loss of work, a confirmation is built into the system.

To keep the local queue intact and return to the previous page, press the [No] button.

To go ahead and delete the entire queue, press the [Yes, Clear Local Queue and Restart Agenda] button.

If you want to keep a record of the operations in the local queue (e.g., to try running them again at some later time), press the [No] button to return to the previous page. Then view the contents of the local queue and copy some or all of the operations to a text file.

Frequently, after clearing the local queue, the next step is to restart the agenda. If you select [Yes, Clear Local Queue and Restart Agenda], the agenda will be restarted once the local queue is gone.
