A clause struc is a knowledge base storage optimization for those cases where the same assertion sentence is asserted or concluded (through forward inference) in different micro-theories.

In those situations, the CYC KB will store the assertion sentence once, inside a clause struc(ture), and reference that clause struc from within the assertion objects.
