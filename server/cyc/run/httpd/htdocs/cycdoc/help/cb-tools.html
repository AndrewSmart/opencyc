This page allows the user to customize the toolbar (the top frame of the browser window), by adding or removing links to the tools listed on this page. It also serves as a launching point to the rest of the tools in the KB Browser.

Changing the Toolbar

To add tools to the Toolbar, select the check box to the left of the tool to be added. To remove a tool, uncheck the relevant box. Then, making sure the buttons by the tools you want displayed in the Toolbar are checked, and the tools you don't need are unchecked, click on the button labeled [Update Toolbar] to initiate the changes. Note that it is not possible to remove "Tools" from the Toolbar Frame.

The [Current Values] button resets the buttons to their values before any changes were made.

To abort out of updating the Toolbar, press the [Cancel] button, which will return you to the previous frame.

Navigating Around the KB Browser

Each of the tools listed on this page is a direct link to that tool. Thus this page can also be used as a "stepping stone" to go to those tools which you don't use often enough to merit its being added to the toolbar. To use a tool, simply click on its highlighted name. Note that not all tools will be available in all versions of the CYC® System, particulary the Wales and WordNet tools at the end of the list.

The Tools

  • Ask: ask CYC® a query in CycL.
  • Assert: assert a single formula in CycL into the KB.
  • Compose: add knowledge to the KB in special KE-Text format.
  • Create: create a new CYC® constant.
  • Doc: go to the CYC® documentation Table of Contents.
  • Hier: use the Hierarchy Browser to see where a given constant fits into various hierarchical listings of constants.
  • History: get a listing of recently viewed constants and assertions.
  • Hypothesize: ask CYC® to hypothesize a formula, creating new constants as needed.
  • Interactor: evaluate SubL expressions in a special evaluation window.
  • Navigator: a semi-complete listing of possible actions/tools/applications in CYC®.
  • Options: set interface preferences for the KB Browser.
  • Overlap: find assertions in the KB which contain two or more specified constants.
  • Viewer: display a listing of local transcripts.
  • Wales: add lexical information to the KB.
  • When: find assertions or constants by the date they entered the KB (Display by Date).
  • WordNet: link terms in CYC® to terms in WordNet, an online dictionary/thesaurus.
