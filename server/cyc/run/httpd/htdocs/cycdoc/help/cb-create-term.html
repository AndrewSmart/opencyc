This page allows the user to create a new CYC® term, with documentation and lexification. For a more spartan interface to create constants, see the Create Constant Help documentation.

Creating New Constants

To create a new constant in the KB, enter the name the new constant will have (minus the "#$"), then press the [Create Constant] button. If a constant already exists by that name, a message will appear to that effect.

To see if any existing constants have similar beginnings, enter the intial part of the name into the window and press the [Complete] button. Possible completions will be displayed in a separate completion window which will be reused for future completions. Completion is case sensitive, and requires that JavaScript is enabled in your browser.

To clear any of the entry fields in the window and start over, press the [Clear] button. To clear all of the entry fields together, press the [Reset All Fields] button.

To abort out of the create constant option and return to the previous frame, press the [Cancel Create] button.

Providing a Comment

For every constant that is created, it is important to provide a descriptive English #$comment about it. The Create Term window provides a multi-line entry field for specifying such a comment. To use the completion window from within the comment's multi-line entry field, use the [Complete] button above the entry field, adjacent to the comment label. To clear the entry field in the window and start over, press the [Clear] button.

Providing Natural Language Information

Constants that are to participate in natural language understanding require lexification. The most straight-forward way of lexification is to provide a term string for a Cyc term. The Create Term window has a multi-line entry field for specifying term strings; each line is interpreted as one term string for the newly created Cyc term. To clear the entry field in the window and start over, press the [Clear] button.

Next Steps

Once a term has been successfully created, provided with a comment and natural language lexification, the term needs to be positioned in the ontology. The Create Term page therefore hands off to the Categorize Term page.
