Viewing Local Transcripts

The Transcript Viewing Area displays a listing of all the local transcripts that have been created for the current local CYC® image. Transcripts whose operations have already been sent to the Master Transcript will be marked with "-sent" just before the ".ts" in their name. For example, operations in the local transcript:

    WILLY-397-19980109095101-local-0-sent.ts
have already been sent, while operations in the transcript: 
    WILLY-397-19980109095101-local-1.ts
have not.

To view a particular transcript in the list, simply click on the name of that transcript. Remember that some of these files might be quite long.

Note that transcripts from past images (even if it's the same KB) will not be listed, nor will local transcripts from other machines. To view these transcripts, go to the Navigator, click "Show Directory" in the File Handling section, and enter the appropriate transcript directory in the input box.

Transcript Statistics

Clicking on the "Stats" link to the left of a local transcript name in the list will bring up a set of statistics for that transcript:

  • Transcript: shows the name of the transcript for which statistics are being provided.
  • Operations: lists the number of operations in the transcript.
  • Constants Created: lists the number of new constants created by operations in the transcript.
  • Constants Renamed: lists the number of constants renamed by opearations in the transcript.
  • Constants Killed: lists the number of constants killed by operations in the transcript.
For more detailed information on CYC® transcripts, see the CYC® Transcript Documentation.
