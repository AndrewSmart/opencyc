The Toolbar Frame allows the user to do three things:

  • Search for constants using the input window.
  • Navigate around the KB Browser via the "Tools" and "Navigator" links.
  • Access the user login facility via the "Login" link.

Toolbar Layout

The layout of the Toolbar depends on whether JavaScript is enabled or not. With JavaScript on, there will be a single, multi-purpose window (for options 1-4 below); CYC® constant completion (described below) will also be possible. With JavaScript off, there will be two windows--one for options 1 and 3 (left-hand window), one for option 4 (right-hand window); CYC® constant completion will not be available (thus option 2 below is not possible), since it requires JavaScript to run.

JavaScript can be turned on and off on the Browser Options page, accessible by clicking on the "Options" link on the Toolbar or Browser Tools page.

JavaScript is not enabled by default because certain HTML browsers do not support it fully. To configure your KB Browser to the capabilities of your HTML browser, go to the CYC® Main Menu (click the "Navigator" link to go to the CYC® Navigator) and click on "Browser Setup". The system will determine if your HTML browser supports JavaScript and frames, and configure your KB Browser accordingly.

Searching for constants in the KB:

There are four ways to search using the input window:

  1. Enter a complete constant name, then press the [Show] button. (Use the left-hand window with JavaScript off.) There are four versions of this search:
    • Entering a constant name without "#$" (e.g. Foo) uses a case-insensitive lookup, with help from the lexicon.
    • Entering a constant name with "#$" (e.g. #$Foo) uses exact, case-sensitive lookup.
    • If an entry is preceded by a left parenthesis (e.g. (#$FooFn #$Bar)), an exact, case-sensitive lookup is used to find the specified reified NAT.
    • If an entry is in double quotes (e.g. "foo bar"), the system uses only the lexicon to find constants. This option is only available in versions of CYC® containing the lexicon code.
  • Enter the first part of a constant name and press the [Complete] button. Then press the [Show] button. Possible completions will be displayed in a separate completion window which will be reused for future completions. Completion is case sensitive, and requires that JavaScript is enabled in your browser. (This option is not available with JavaScript turned off.)
  • Enter a substring of a constant name, using "*" as a wildcard at the tail end of the string (or at both ends) to represent unspecified characters in the string. Then press the [Show] button. Substrings must be at least 3 characters long for this to work. (Use the left-hand window with JavaScript off.)
  • Enter a UNIX regular expression and press the [GREP] button. Clicking on the "Ignore Case" button causes the GREP search to become case-insensitive, unclicking makes it sensitive to case. The default setting is to ignore case. In general, the GREP search is faster than the wildcard method mentioned above. However, since the constant search list for the GREP facility is only updated when a new CYC® world is created, the wildcard method may reflect changes to constant names faster than the GREP facility. For help with UNIX regular expressions, click here. (Use the right-hand window with JavaScript off.) If any of the above searches yields more than one possible constant, a list of constants will be displayed. Clicking on the desired constant will display the entry for that constant.

Navigating Around the KB Browser

To reach other tools accessible from the KB Browser, click on the "Tools" link. This will take you to the Browser Tools page, where you can either continue on to another tool, or modify the Browser Toolbar so that links to desired tools appear directly in the Toolbar Frame, thus becoming more readily accessible.

The help page page for the Browser Tools explains each of the available tools in more detail.

The CYC® Navigator is a useful launching pad for accessing the rest of the CYC® System. To reach it, simply click on the "Navigator" link on the Toolbar or Browser Tools pages.

Interface preferences for the KB Browser can be set by going to the Browser Options page, accessible via the "Options" link on the Toolbar or Browser Tools pages.

Login

In order to fully access all areas of the KB Browser, it is necessary to first login as a CYC® user. The "Login" link in the Toolbar lists the current user, and connects to the Login Area, where new users can login to CYC®.

Other Information

Clicking on "Update" refreshes the Toolbar Frame. This link will not appear if JavaScript is turned off. If frames are not being used, this link will also not appear; use the "Update" link in the Agenda Status Bar to refresh the entire page.

"Machine" lists the machine on which the CYC® Process is running.
