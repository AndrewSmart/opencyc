Viewing a CycL Sentence

This page displays a CycL sentence. This sentence might or might not correspond with any assertion in the KB. It might be true, false, or unknown.

CycL Sentence Details

  • "Mt" lists the #$Microtheory of this sentence, if it is a contextualized sentence. A contextualized sentence is a sentence of the form (#$ist MT PROP).
  • "HL Formula" lists the actual sentence being viewed, in HL form.
  • "EL Formula" lists the sentence in its EL form, if the HL form differs from the EL form. Many CycL sentences have identical HL and EL forms.
  • "English Translation" provides an English version of the assertion, generated by CYC®.

Operations on CycL Sentences

There are a number of different operations involving CycL sentences, many of which are similar to operations on assertions.

  • "Diagnose" performs well-formedness (wff) checking on the CycL sentence.
  • "Assert Similar" brings up an editing window with the CycL sentence as a default. The user can then modify the new copy (or not) before asserting it into the KB.
  • "Ask Similar" brings up an editing window with the CycL sentence as a default. The user can then modify the new copy (or not) before asking it.
For more information about CycL sentences, read the Syntax of CycL.
