This page allows the user to find an assertion in Cyc®.

To find an assertion, enter the assertion sentence into the "EL Sentence" box and press [Find Assertion]. If the sentence is asserted in an accessible microtheory, it (and the microtheory in which it is asserted) will be displayed.
