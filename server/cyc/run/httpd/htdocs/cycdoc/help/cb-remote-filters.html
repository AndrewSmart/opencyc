This page shows currently active filters and allows the user to make changes to the current filtering state.

Saving and Loading Filter Settings

Filter settings can be saved out to a file. There is a unique file for each Cyclist, so saving user settings will overwrite previous saved settings for that Cyclist.

Once saved, filter settings can be loaded back in from a file. The file loaded will be the unique file of settings for the cyclist logged in.

Methods of Filtering Operations

Remote operations can be filtered by specifying any of three objects, as follows.
  • Filter Terms: Specifying a term to use will cause all remote operations which contain that term to be ignored by the CYC® Agenda. That means that those operations will not be loaded into the CYC® image and will not be accessible thereafter. Note NATs are not explored recursively to find all atomic terms which make them up. This means that a CYC® formula will not be filtered for containing a NAT which contains a filtered term.
  • Filter Collections: Specifying a collection to use will cause all remote operations which contain any spec of that collection to be ignore by the CYC® Agenda exactly as if that spec had itself been listed as a term to be filtered.
  • Transcript Filters: Specifying a transcript filter to use will cause all terms associated with the transcript filter via the #$filterTerms predicate to be filtered exactly as if they had been specified separately as above. Similarly all collections associated with the transcript filter via the #$filterCollections predicate will be filtered as filter collections above.

Adding and Removing Filter Settings

Filter settings can be added to any of the three categories available by simply typing in the name of the term, collection, or transcript filter in the appropriate box and clicking on the "Update Filters" button at the top of the page. Clicking "Reset Values" will clear all boxes of whatever text has been typed in them.

To remove filter settings simply uncheck the checkbox beside the CYC® constant that you do not want filtered and then click the "Update Filters" button. Clicking "Remove All Filters" will remove all the active transcript filters, collections and terms. This is equivalent to unchecking them all and then clicking "Update Filters".
