This page allows the user to minimally position a newly created CYC® term in the Cyc ontology by assigning it to one collection that the term is an instance of, and (so applicable) one collection that the term specializes.

Press the [Categorize] button when all applicable fields have been filled in.

To clear any of the entry fields in the window and start over, press the [Clear] button. To clear all of the entry fields together, press the [Reset All Fields] button.

The Term

The page shows the term undergoing categorization at the top, together with its comment.

Note: Existing terms can be re-categorized, but this page does not show similar terms, instance or specialization information already asserted about this term.

Mt

Enter the microtheory (minus the "#$") in which the categorizations are to be asserted. To take advantage of constant completion, enter a partial microtheory name and press the [Complete] button. Remember that a list of completions will appear in a separate HTML browser window if the completion is not unique.

Similar To

Use this field to enter a term (minus the "#$") that is similar to the term being categorized. To take advantage of constant completion, enter a partial term name and press the [Complete] button. Remember that a list of completions will appear in a separate HTML browser window if the completion is not unique.

Instance Of

Use this field to enter the collection (minus the "#$") of which this term is an instance. All terms must be at minimum instances of #$Thing. To take advantage of constant completion, enter a partial term name and press the [Complete] button. Remember that a list of completions will appear in a separate HTML browser window if the completion is not unique.

Specialization Of

Use this field to neter the collection (minus the "#$") that this term specializes, if the term is itself a collection. All collections must be at minimum specializations of #$Collection. To take advantage of constant completion, enter a partial term name and press the [Complete] button. Remember that a list of completions will appear in a separate HTML browser window if the completion is not unique.

Next Steps

If similarity information is provided, the Categorize Term leads to the Copy from Similar page.

If no similarity information is provided, the Categorize Term page leads to the Conceptually Related Terms page.
